Website Facebook Pixel

Adds Facebook Pixel to website pages.

How to use this module

CONFIGURATION

To configure, follow next steps:

  • Go to the Website Admin - Configuration - Settings menu.
  • Enter Facebook Pixel Key.
  • Apply the changes.
  • The Facebook Pixel will be activated on all pages of the website.
