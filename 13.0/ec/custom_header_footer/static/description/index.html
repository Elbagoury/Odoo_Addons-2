Custom Header Footer

Overview

  Features

  • This module is display a custom header footer on report as per selected options in Tradmark field on sale,invoice and stock records.
  • User can use 4 different Header and Footer combination
  • Print Report diffrant type of header and footer on Crm ,Saleorder, Stock, and invoice.

Add 4 different Header and 4 different footer on company.

Select required options in Tradmark field. According option selection you will get Header/Footer of report.

Selected company is display in sale order

Generate report Header/Footer as per set images on company
