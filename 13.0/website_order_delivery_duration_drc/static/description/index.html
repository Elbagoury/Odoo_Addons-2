Customer message and wanted delivery date during checkout.

User can add delivery date & leave a message during checking out from website.User can't select past date.If user add's past date then it will display error message and user can't proceed further.

Delivery date & message in sale order and report of sale order, which were dropped by user on website during checkout.

If you make delivery order from sale order, then it will transfer delivery date and delivery message from sale order to delivery order as well as report of delivery order.

If you make invoice directly from sale order or after confirming the delivery order, then it will transfer message from sale order to invoice as well as report of invoice.
